This is a fully responsive and interactive personal portfolio website for **Sakshi Udakhe**, showcasing her skills, projects, and contact information.

## Features

### 1. **Header Section**
- Displays a logo and navigation links.
- Responsive navigation menu with a hamburger toggle for mobile devices.

### 2. **Home Section**
- Introduction to Sakshi Udakhe, including roles like Frontend Developer, Software Tester, and Freelancer.
- Animated text with a changing word effect.
- Social media icons for external linking.

### 3. **About Section**
- Information about Sakshi Udakhe as a web developer.
- Aligned content with a left-positioned profile image.

### 4. **Skills Section**
- **Technical Skills:** Progress bars for HTML, CSS, JavaScript, React, and Bootstrap.
- **Professional Skills:** Circular skill representations for teamwork, problem-solving, communication, and creativity.

### 5. **Projects Section**
- Filters for project types (All, Internship, College).
- Details for each project with descriptions, images, and external links.

### 6. **Contact Section**
- A contact form to submit user details, including name, email, phone number, and message.

### 7. **Footer Section**
- Copyright text.
- Scroll-to-top button.

## Technologies Used
- **HTML5** for structure.
- **CSS3** for styling and animations.
- **JavaScript** for interactivity and functionality.
- **Boxicons** for icons.
- **Mixitup.js** for project filtering.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/portfolio.git
   ```
2. Open the project directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your browser.

## Responsive Design
The website is designed to adapt seamlessly to different screen sizes:
- **Large Screens:** Full-width layout with properly spaced sections.
- **Medium Screens:** Two-column layout for sections like skills and projects.
- **Small Screens:** Stacked sections with a hamburger menu for navigation.
